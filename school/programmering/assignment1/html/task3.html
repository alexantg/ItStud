Comment:
When you dont convert the input values to a number the addition becomes wrong. The + operator can also be used to put two text-strings together. Because the input values are read as strings before you convert them into numbers the program will take the values and just combine them. For example: 2+2=22. For the other mathematical operations as for example multiplying Javascript understands that the values are numbers once you use the * operator. For all of the mathematical operations to work properly you have to convert the input values to numbers before you use them in a mathematical operation. This can be done using for example parseInt. This converts the values to an integer. If you want to include decimal-numbers you can use parseFloat.
